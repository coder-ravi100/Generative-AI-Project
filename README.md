# LSTM Text Generation using TensorFlow

## Overview
This project implements a character-level text generation model using LSTM networks.
The model is trained on Shakespeare's complete works and generates new text based on seed input.

## Dataset
- Source: Project Gutenberg
- Link: https://www.gutenberg.org/ebooks/100
- Format: Plain Text UTF-8

## Model Architecture
- Embedding Layer
- Two LSTM Layers
- Dense Softmax Output Layer
